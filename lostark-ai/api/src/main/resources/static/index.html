간단한 테스트 채팅 화면

브라우저에서 바로 질문을 보내고 응답을 확인할 수 있습니다. API key는 브라우저 로컬 스토리지에만 저장되며 서버에는 요청할 때만 함께 전송됩니다.

Connection

입력하지 않으면 서버에 설정된 기본 API key를 사용합니다.

테스트 팁

  • 응답 실패 시 메시지 영역에 quota 또는 인증 오류가 그대로 표시됩니다.
  • 현재 화면은 간단한 JSON 채팅 테스트용입니다.
  • 루트 경로 `/` 에서 바로 열립니다.
