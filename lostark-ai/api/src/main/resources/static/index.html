  • 경제 이슈는 패치 기대와 붙을 때 폭발합니다
    단순한 골드 시세보다 상재, 보석, 배럭 효율처럼 실제 지출 의사결정에 가까운 키워드가 더 오래 남습니다.
  • 직업 이슈는 댓글에서 더 선명합니다
    제목만 보면 밸런스 토론 같아도, 댓글을 같이 보면 체감 딜, 레이드 취업, 유저 피로도처럼 더 구체적인 포인트가 드러납니다.
  • 그래서 메인 화면은 워드클라우드보다 랭킹형이 낫습니다
    유저는 예쁜 그림보다 “지금 뭐가 왜 뜨는지”를 바로 읽고 싶어하기 때문에 대표 글과 상승률을 붙이는 편이 훨씬 실용적입니다.
